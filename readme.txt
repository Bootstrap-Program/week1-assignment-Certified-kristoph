
******************
The project is a Gradle project which uses Gradle Wrapper
The dependencies are also managed by Gradle.

Pre-requisite:
# Java installed and path set in System Variable (as JAVA_HOME) 

To Run
1. Unzip the files(if compressed)
2. Go to top level directory of the project (wordCount) - you should see "src", "gradlew" here.
3. Execute this command in commandline ./gradlew --refresh-dependencies - To refresh all dependencies
4. Execute this command in commandline ./gradlew test - To run the tests
